This release prepares DividendTracker as an open-source project.

## Added

- MIT License
- Expanded README
- Contributor guide
- Security policy

## Notes

DividendTracker is an early-stage local-first mobile app for tracking Japanese and US dividend portfolios. It is built with Expo, React Native, TypeScript, and expo-router.
